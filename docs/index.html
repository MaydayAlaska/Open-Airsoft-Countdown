Open Airsoft Countdown

Installa il firmware ufficiale sull’ESP32-S3 direttamente dal browser, senza Visual Studio Code e senza compilare il sorgente.

Questo browser non supporta l’installazione tramite porta seriale. Usa un browser aggiornato compatibile con Web Serial su un computer. L’installer richiede una connessione HTTPS.

Come installare

  1. Collega la scheda ESP32-S3-DevKitC-1 N8 al computer con un cavo USB dati.
  2. Premi Installa il firmware.
  3. Seleziona la porta seriale dell’ESP32-S3.
  4. Conferma l’installazione e attendi il completamento.
  5. Al primo avvio usa il PIN amministratore predefinito 000000.
Durante un aggiornamento, scegli di mantenere i dati per conservare configurazione e utenti. Esegui invece una cancellazione completa per una nuova installazione pulita.
